# InputTable

Install and run
```bash
npm install
node server.js
```

Add a `config.json` file in `config/` folder like this
```js
{
    "user": "example_user",
    "password": "example_password",
    "server": "EXAMPLE_SERVER",
    "database": "example_db",
    "port": 1433,
    "options":
    {
        "instanceName": "EXAMPLE_INSTANCENAME"
    }
}
```
